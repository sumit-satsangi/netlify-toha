    It is always good to clear the concepts first and then take baby steps to execute what you’ve learnt. This is going to be in parts and I will share some basic concepts behind R here. Let’s dive in.

    Programming Styles in R

    There are three main programming styles that R uses:

    • Object Oriented Programming
    • Array Programming &
    • Functional Programming

    Let’s look into them in detail:

    Object Oriented Programming
